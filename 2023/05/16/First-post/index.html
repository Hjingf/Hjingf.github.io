Hexo+github搭建个人博客博客

本文最后更新于：2023年5月24日 晚上

其实从在大学的时候我就想做自己的博客网站了，但是当时以为只有自己买一个服务器才能搭建网站，所有当时一直没有下定决心。工作之后了解到可以通过Hexo+githubpages的方式搭建博客，所以我就果断用这种方式搭建了。以下是我记录的搭建的过程。

1.准备工作

1.1 需要下载的工具

​ Node.js、npm、git
​ 我是先在linux下先试验了一下，然后再在windows下部署的，所以以下的准备工作windows和linux都有包含

1.1.1windows

直接下载安装就可以了

查看版本（是否安装成功）

node -v
npm -v
git –version

1.1.2linux(ubuntu)

​ linux下下载就比较方便了，使用命令下载就可以了

1.2 git账号

git地址：Github

右上角有个sign up，点进去注册，记住用户名

打开gitbash，设置用户名和邮箱

创建SSH密钥

所生成文件在[C:\Users\用户名.ssh] 目录，查看id_rsa.pub公钥内容并复制。

添加公钥到github： Setting-> SSH and GPG keys-> New SSH key

验证链接：

之后创建一个仓库，格式为【你的用户名 + github.io】。

之后浏览器访问 https://<GitHub 用户名>.github.io，如果能正常访问，表示Github Pages搭建成功。
